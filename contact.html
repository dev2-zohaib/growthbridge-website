Testing notes

The form submits JSON to window.GROWTHBRIDGE_CONFIG.contactEndpoint. The default value points to a mock endpoint and can be changed in assets/js/config.js.

  • Front-end validation runs before submission.
  • Failed requests display a clear fallback message.
  • A Postman collection is included in postman/GrowthBridge-Contact-API.postman_collection.json.
